# Agro Forte e Futuro Sustentável

Este projeto é um site desenvolvido em HTML e CSS com o objetivo de apresentar a importância do agronegócio e da sustentabilidade para o futuro. O site destaca como o agro contribui para a produção de alimentos, a geração de empregos e o crescimento econômico, além de mostrar a importância da preservação ambiental.

## Conteúdo do site

* **Agro Forte:** explica a relevância do agronegócio para a sociedade e a economia.
* **Sustentabilidade:** apresenta práticas que ajudam a proteger os recursos naturais.
* **Tecnologia e Inovação:** mostra como novas tecnologias aumentam a produtividade no campo.
* **Futuro Sustentável:** destaca a necessidade de equilibrar desenvolvimento econômico e preservação ambiental.

## Tecnologias utilizadas

* HTML5
* CSS3
* Visual Studio Code

## Objetivo

Conscientizar sobre a importância de um agronegócio moderno, eficiente e sustentável, garantindo alimentos de qualidade e a preservação do meio ambiente para as futuras gerações.
